# Comrades 2027 — Training Dashboard

A self-contained training tracker for the 100th Comrades Marathon (Down Run, ~89 km).

- 52-week plan (gym Mon/Wed/Fri, run Tue/Thu/Sat) with recommended paces
- Log each run (distance, time, effort, elevation, HR) → instant per-run feedback
- An autoregulation "coach" that adjusts next week's targets from your real data
- All data stays **private in your own browser** (localStorage) — nothing is uploaded

**Open it:** https://mdcockcroft.github.io/comrades-dashboard/
Add it to your phone's home screen for an app-like experience.
